Question number one?

It depends on the condition of the tooth. If it contains a cavity a reduction in the price will have to be negotiated with the fairy and this will need to be done whilst you are in slumber so this can prove difficult. Your behaviour will also be a deciding factor so it pays to be good. Current TPI (Tooth Price Index) is around 80p for a molar and 69p for an incisor.
